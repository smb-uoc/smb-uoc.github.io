Fuentes de Información

En nuestro sitio web, nos esforzamos por proporcionar contenido preciso y relevante. A continuación, encontrarás algunas de las fuentes de información que utilizamos para recopilar datos, noticias y detalles sobre la banda:

Atribuciones y Derechos

Para garantizar el cumplimiento de los derechos de autor y copyleft, se atribuyen los derechos correspondientes a las fuentes originales de las imágenes, videos y contenido textual utilizados en este sitio web.

Las imágenes y videos utilizados han sido obtenidos de fuentes públicas, como el sitio oficial de la banda y otros recursos en línea relacionados con Red Hot Chili Peppers.

Herramientas de IA Generativa

Queremos hacer constar que hemos utilizado las siguientes herramientas de inteligencia artificial generativa:

  • Copilot: Se utilizó para generar algunas imágenes y gráficos presentes en el sitio web.
  • ChatGPT: Se empleó para generar parte del contenido textual del sitio web, como descripciones y texto explicativo.

Componentes de Terceros embebidos

Reproductor de Spotify

El reproductor de Spotify embebido se utiliza para permitir a los usuarios escuchar música de Red Hot Chili Peppers directamente en el sitio web. Se accede a través de enlaces específicos en las páginas relacionadas con la música de la banda.

Videos de YouTube

Los videos embebidos de YouTube se utilizan en varias páginas del sitio web para mostrar presentaciones en vivo, entrevistas y otros contenidos relacionados con Red Hot Chili Peppers.

Referencias Técnicas

Las tecnologías utilizadas en el desarrollo de este sitio web son las siguientes:

  • HTML
  • CSS
  • JavaScript (JS)
  • Parcel
  • Node.js
  • NPM (Node Package Manager)
  • Dependencias Externas (Desarrollo):
    • W3C HTML Validator: Utilizada para validar los ficheros HTML
    • CSS Validator: Utilizada para validar los ficheros CSS
  • Dependencias Externas (Productivas):
    • Tilt.js: Utilizado para los efectos 3D en las imágenes.
    • Lyrics API: Utilizada para la búsqueda de letras de canciones

Es importante destacar que todas las herramientas y tecnologías se utilizaron de manera ética y responsable en el desarrollo del sitio web.
